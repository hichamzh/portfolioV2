## ⚙️ Installation

1. 📥 Clonez :
   ```bash
   git clone https://github.com/votre-utilisateur/portfolio.git
   ```

2. 📂 Accédez :
   ```bash
   cd portfolio
   ```

3. 📦 Installez :
   ```bash
   npm install
   ```

4. 🚀 Lancez :
   ```bash
   npm run dev
   ```

5. 🌐 Ouvrez : [http://localhost:5173](http://localhost:5173).
